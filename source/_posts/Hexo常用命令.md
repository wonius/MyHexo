## 创建新文档

```shell
hexo new filename
```

## 启动本地服务

```
#启动本地服务，然后访问http://localhost:4000/
hexo s
```

## 重新编译并且发布

```
hexo g && hexo d
```

## 清理缓存

```
hexo clean
```
